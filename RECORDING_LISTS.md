# Recording Lists for Young Israel

This document contains all audio recordings needed for Aleph2Davening app.

## 1. TEFILLOS (PRAYERS) TO RECORD

Record each prayer as a **complete, clear reading** with nekudot (vowel points). Record at a **moderate pace** suitable for beginners to follow along.

### Morning Prayers (Shacharit)
1. **Modeh Ani** (מוֹדֶה אֲנִי) - Said upon waking
2. **Netilat Yadayim** - Blessing for hand washing
3. **Asher Yatzar** - Blessing after using the bathroom
4. **Elokai Neshama** - Blessing thanking God for the soul
5. **Birchos HaTorah** - Blessings before learning Torah
6. **Birchos HaShachar** - Morning blessings
7. **Baruch She'amar** - Opening blessing of Pesukei D'Zimra
8. **Hodu** - Thanksgiving psalm (Psalm 107)
9. **Mizmor L'Todah** - A psalm for thanksgiving (Psalm 100)
10. **Ashrei** - Happy are those (Psalm 145)
11. **Az Yashir** - Song at the Sea
12. **Yishtabach** - Closing of Pesukei D'Zimra
13. **Yotzer Or** - Blessing before Shema (morning)
14. **Ahavah Rabbah** - Second blessing before Shema
15. **Shema** - Hear O Israel (Deuteronomy 6:4-9)
16. **V'ahavta** - And you shall love (part of Shema)
17. **Emet V'Yatziv** - Blessing after Shema
18. **Shemoneh Esrei (Amidah)** - Silent prayer (weekday version)
19. **Tachanun** - Supplication prayer
20. **Uva L'Tzion** - A redeemer will come to Zion
21. **Aleinu** - It is our duty
22. **Ein Kelokeinu** - There is none like our God

### Shabbat Prayers
23. **Kabbalat Shabbat** - Welcoming Shabbat psalms
24. **Lecha Dodi** - Come my beloved
25. **Shabbat Candle Lighting** - Blessing for candles
26. **Kiddush (Friday Night)** - Sanctification over wine
27. **Kiddush (Shabbat Day)** - Morning kiddush
28. **Havdalah** - Separation ceremony (Saturday night)

### Holiday Prayers
29. **Hallel** - Psalms of praise (for holidays)

### Brachot (Blessings on Food/Drink)
30. **HaMotzi** - Blessing on bread
31. **Mezonot** - Blessing on grains
32. **HaGafen** - Blessing on wine/grape juice
33. **HaEtz** - Blessing on fruit from trees
34. **HaAdamah** - Blessing on vegetables
35. **Shehakol** - Blessing on everything else
36. **Birkat HaMazon** - Grace after meals
37. **Borei Nefashot** - After-blessing for most foods
38. **Al HaMichya** - After-blessing for grain products

**TOTAL PRAYERS: 38**

### Recording Instructions for Prayers:
- Record in a quiet room with no background noise
- Use moderate pace (not too fast, not too slow)
- Clearly enunciate each syllable
- Follow traditional Ashkenazi or Sephardi pronunciation consistently
- Each prayer should be ONE complete audio file
- File naming: `prayer-name-youngisrael.mp3` (e.g., `modeh-ani-youngisrael.mp3`)

## 2. ALEPH BET (HEBREW LETTERS) TO RECORD

Record each letter's **sound** (NOT the letter name). For example:
- Aleph is silent (carry the vowel)
- Bet sounds like "B"
- Shin sounds like "SH"

### All 22 Letters (Regular Forms)
1. **Aleph** (א) - Silent
2. **Bet** (בּ) - "B"
3. **Vet** (ב) - "V" (same letter as Bet, no dagesh dot)
4. **Gimmel** (ג) - "G" (like "go")
5. **Dalet** (ד) - "D"
6. **Hei** (ה) - "H"
7. **Vav** (ו) - "V"
8. **Zayin** (ז) - "Z"
9. **Chet** (ח) - "CH" (guttural, like Bach)
10. **Tet** (ט) - "T"
11. **Yud** (י) - "Y"
12. **Kaf** (כּ) - "K"
13. **Chaf** (כ) - "CH" (same letter as Kaf, no dagesh)
14. **Lamed** (ל) - "L"
15. **Mem** (מ) - "M"
16. **Nun** (נ) - "N"
17. **Samech** (ס) - "S"
18. **Ayin** (ע) - Silent (guttural stop)
19. **Pei** (פּ) - "P"
20. **Fei** (פ) - "F" (same letter as Pei, no dagesh)
21. **Tzadi** (צ) - "TZ"
22. **Kuf** (ק) - "K"
23. **Resh** (ר) - "R"
24. **Shin** (שׁ) - "SH"
25. **Sin** (שׂ) - "S" (same letter as Shin, different dot)
26. **Tav** (ת) - "T"

### Final Forms (5 letters)
27. **Final Chaf** (ךְ) - "CH"
28. **Final Mem** (ם) - "M"
29. **Final Nun** (ן) - "N"
30. **Final Pei** (ף) - "F"
31. **Final Tzadi** (ץ) - "TZ"

**TOTAL LETTERS: 31 sounds**

### Recording Instructions for Letters:
- Record ONLY the sound, not the name (e.g., for Shin, say "SH" not "shin")
- Keep it short - 0.5 to 1 second per letter
- Clear pronunciation
- File naming: `letter-name.mp3` (e.g., `shin.mp3`, `aleph.mp3`)

## 3. VOWELS (NEKUDOT) TO RECORD

Record each vowel sound clearly.

1. **Kamatz** (like ָ ) - "AH" (as in "father")
2. **Patach** (like ַ ) - "AH" (same sound as Kamatz)
3. **Segol** (like ֶ ) - "EH" (as in "bed")
4. **Tzere** (like ֵ ) - "EH" (same as Segol, longer)
5. **Chirik** (like ִ ) - "EE" (as in "see")
6. **Cholam** (like ֹ ) - "OH" (as in "boat")
7. **Kubutz** (like ֻ ) - "OO" (as in "boot")
8. **Shuruk** (וּ) - "OO" (same as Kubutz)
9. **Shva** (like ְ ) - "uh" (very short, like "about")

**TOTAL VOWELS: 9 sounds**

### Recording Instructions for Vowels:
- Record just the vowel sound, no consonant
- Keep it clear and sustained (about 1 second)
- File naming: `vowel-name.mp3` (e.g., `kamatz.mp3`)
